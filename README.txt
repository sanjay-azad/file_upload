Requirements:
python 3.5 or above

Library:
django 2.*
(CMD -> pip install django)

To run the app:
In CMD, go to the project directory -> 
type: python manage.py runserver 0.0.0.0:8002
(python should be in the environment variable. Or else replace python with full location of python.exe)
